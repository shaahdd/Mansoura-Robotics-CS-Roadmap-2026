## 📌 About the Circle

The **CS Circle** at *Mansoura Robotics* is a structured technical program designed to build strong foundations in programming and problem solving.

### 🔁 Learning Flow

> **C++ → Object-Oriented Programming → Data Structures**
> From basic syntax → abstraction → efficient problem solving
